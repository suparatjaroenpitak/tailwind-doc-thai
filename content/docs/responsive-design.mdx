## Mobile-first

Tailwind ใช้แนวคิด mobile-first ค่า class ที่ไม่ใส่ prefix จะใช้กับทุกหน้าจอ ส่วน prefix เช่น `sm:`, `md:`, `lg:` จะเริ่มทำงานเมื่อหน้าจอกว้างขึ้น

```html
<div class="grid gap-4 md:grid-cols-2 lg:grid-cols-3">
  <article class="rounded-lg border p-4">A</article>
  <article class="rounded-lg border p-4">B</article>
  <article class="rounded-lg border p-4">C</article>
</div>
```

## ตัวอย่าง responsive card

```html
<section class="grid gap-6 md:grid-cols-[240px_1fr]">
  <aside class="rounded-lg bg-slate-100 p-4">เมนู</aside>
  <main class="rounded-lg border p-6">เนื้อหา</main>
</section>
```

## อธิบายทีละบรรทัด

1. `grid` เปิด CSS Grid
2. `gap-6` เว้นระยะระหว่างช่อง
3. `md:grid-cols-[240px_1fr]` เมื่อถึงขนาด `md` ให้ sidebar กว้าง 240px และ content กินพื้นที่ที่เหลือ
4. ถ้าหน้าจอเล็กกว่า `md` layout จะเรียงลงแนวตั้งโดยอัตโนมัติ

## เทียบกับ CSS ปกติ

```css
.layout {
  display: grid;
  gap: 1.5rem;
}

@media (min-width: 768px) {
  .layout {
    grid-template-columns: 240px 1fr;
  }
}
```

Tailwind ย้าย breakpoint มาอยู่ใกล้ element ทำให้เห็น behavior ของ component ได้ในที่เดียว
