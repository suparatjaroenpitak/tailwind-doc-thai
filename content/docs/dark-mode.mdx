## หลักการ

Dark mode ใน Tailwind ใช้ variant `dark:` นำหน้า utility ที่ต้องการเปลี่ยนเมื่อเว็บอยู่ในธีมมืด เช่น `dark:bg-slate-950` หรือ `dark:text-white`

```html
<div class="bg-white text-slate-900 dark:bg-slate-950 dark:text-white">
  อ่านสบายทั้งโหมดสว่างและโหมดมืด
</div>
```

## ตัวอย่าง

```html
<article class="rounded-lg border border-slate-200 bg-white p-6 dark:border-slate-800 dark:bg-slate-950">
  <h2 class="text-xl font-bold text-slate-950 dark:text-white">Dark mode</h2>
  <p class="mt-3 text-slate-600 dark:text-slate-300">ปรับสีข้อความและเส้นขอบให้ contrast พอดี</p>
</article>
```

## ข้อควรระวัง

อย่าแค่กลับสีขาวเป็นดำทุกจุด ให้ดู contrast ของข้อความ เส้นขอบ และ background ด้วย สีเทาอย่าง `slate` มักเหมาะกับ documentation เพราะไม่แสบตาและอ่านโค้ดได้นาน

## เทียบกับ CSS ปกติ

```css
.card {
  background: white;
  color: #0f172a;
}

.dark .card {
  background: #020617;
  color: white;
}
```

Tailwind:

```html
<div class="bg-white text-slate-900 dark:bg-slate-950 dark:text-white">
  ...
</div>
```
